# Personal Expense Tracker

This is a Python project that allows users to track expenses, manage category-based budgets, search expense records, and visualize spending with a bar chart.

The project was developed to practice Python fundamentals, file handling, input validation, dictionaries, functions, and basic data visualization using turtle.

## Technologies Used

- Python
- turtle
- File handling

## Features

- Add expenses with date, amount, category, and description
- View saved expenses
- Search expenses by keyword
- View expenses by category
- Add and update category budgets
- Show budget alerts when spending exceeds the budget
- Display a bar chart of expenses by category

## Concepts Practiced

- Functions
- Loops
- Conditional statements
- Dictionaries
- File reading and writing
- User input validation
- Basic data visualization
- Menu-based console application structure

## Required Files

This project uses text files to store data:

- expenses.txt
- budget.txt

If these files do not exist, create empty files with these names in the same folder as the Python file.
